# Nexa-Byte Hero Section

A modern hero section built with React, Vite, and Tailwind CSS, inspired by OpenAI's design. 

## Quick Start

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```

4. **Preview production build:**
   ```bash
   npm run preview
   ```
